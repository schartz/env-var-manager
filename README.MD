# envmgr

A lightweight CLI tool written in Zig for interactive environment variable and secret management.
This project is my "learn zig by making things" project.

## Prerequisites

* **Zig:** Version `0.16.0` or higher.

## Usage

To compile and run the tool interactively:

```bash
zig build run

```

To build a release executable (optimized for speed):

```bash
zig build -Doptimize=ReleaseFast

```

The compiled binary will be available in the `zig-out/bin/` directory.
